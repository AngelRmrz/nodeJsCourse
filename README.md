# Plan de Estudio para Aprender Node.js y MongoDB

Este es un plan de estudio diseñado para desarrolladores frontend que desean convertirse en desarrolladores fullstack, aprendiendo Node.js y MongoDB. El plan está estructurado para ser completado en una hora al día.

## Objetivos

- Dominar los fundamentos de Node.js y su ecosistema.
- Aprender a trabajar con MongoDB y realizar operaciones CRUD.
- Desarrollar un proyecto fullstack utilizando Node.js, Express.js y MongoDB.

## Estructura del Plan

### Semana 1: Introducción a Node.js

- **Día 1**: Fundamentos de Node.js y su arquitectura basada en eventos.
- **Día 2**: Instalación y configuración de Node.js.
- **Día 3**: Uso de módulos nativos de Node.js (`http`, `fs`, `path`).
- **Día 4**: Creación de módulos personalizados.
- **Día 5**: Introducción al asincronismo y uso de callbacks.

### Semana 2: Profundizando en Node.js

- **Día 6**: Manejo de errores y excepciones.
- **Día 7**: Uso de streams y buffers.
- **Día 8**: Introducción a Express.js.
- **Día 9**: Creación de rutas y manejo de peticiones HTTP.
- **Día 10**: Uso de middleware en Express.js.

### Semana 3: Bases de datos y Node.js

- **Día 11**: Introducción a bases de datos NoSQL y MongoDB.
- **Día 12**: Instalación y configuración de MongoDB.
- **Día 13**: Estructura de una base de datos MongoDB.
- **Día 14**: Uso del driver oficial
